You are enhancing an existing nutrition tracking web app built with React (frontend) and a backend (Django/Node).

Your goal is to implement a **User Profile + Personalized Goals System + Smart Nutrient Targets + Calorie Burn Suggestions Tab**.

---

## 🧩 FEATURE 1: USER PROFILE (MANDATORY INPUT)

Create a user profile system where each user must enter:

* Age (integer)
* Gender (male/female/other)
* Height (cm)
* Weight (kg)
* Activity Level:

  * sedentary
  * light
  * moderate
  * active
* Goal:

  * Lose weight
  * Maintain weight
  * Gain weight

---

## 🔥 FEATURE 2: DAILY CALORIE GOAL

Use Mifflin-St Jeor Equation:

Men:
BMR = 10×weight + 6.25×height − 5×age + 5

Women:
BMR = 10×weight + 6.25×height − 5×age − 161

Apply activity multiplier:

* sedentary → 1.2
* light → 1.375
* moderate → 1.55
* active → 1.725

Adjust based on goal:

* lose → −300 to −500 kcal
* gain → +300 to +500 kcal
* maintain → no change

Store and display daily calorie target.

### 🍞 CARBS & 🧈 FATS

After protein is calculated:

Remaining calories = total_calories − protein_calories

Split remaining:

* 60% → carbs
* 40% → fats

Convert:

* carbs: 1g = 4 kcal
* fats: 1g = 9 kcal

Store and show:
